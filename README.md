# NLP for Twitter Sentiment Analysis
Logistic regression's implementation for sentiment analysis on tweets. Given a tweet, it will decide if it has a positive sentiment or a negative one.

- Tensorflow 2.3.1
- Keras
- numpy 1.18.0
- Docker
